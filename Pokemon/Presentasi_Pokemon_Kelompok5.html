1. Gambaran Umum Proyek

Proyek ini adalah clone game Pokemon yang dibuat menggunakan LibGDX framework untuk Java. Game ini menerapkan pola arsitektur MVC (Model-View-Controller) dengan fitur-fitur lengkap seperti:

  • Eksplorasi dunia dengan tile-based map
  • Sistem pertarungan Pokemon turn-based
  • Sistem save/load game
  • Dialog dan interaksi NPC
  • Animasi karakter (berjalan, berlari, bersepeda)

Step 2: Game Initialization (PokemonGameMain.java)

public void create() {
    // 1. Inisialisasi managers
    assetManager = new AssetManager();
    tweenManager = new TweenManager();
    moveDatabase = new MoveDatabase();
    saveManager = new SaveManager();
    
    // 2. Load shaders untuk efek visual
    initShaders();
    
    // 3. Load semua assets
    loadAssets();
    assetManager.finishLoading();
    
    // 4. Create dan show GameScreen
    gameScreen = new GameScreen();
    gameScreen.init(this);
    setScreen(gameScreen);
}

Step 3: GameScreen Activation (GameScreen.java)

public void show() {
    // 1. Setup player
    world = worlds.get("littleroot_town");
    player = new PlayerActor(world, safeX, safeY, animations, this);
    world.addActor(player);
    
    // 2. Load atau new game
    if (saveManager.hasSaveGame(0)) {
        // Load saved data
    } else {
        // New game - beri starter Pokemon
        playerTrainer = new Trainer(
            Pokemon.generatePokemon("Bulba", "bulbasaur.png", moveDatabase)
        );
    }
    
    // 3. Setup controllers
    playerController = new ActorMovementController(player);
    worldRenderer = new WorldRenderer(assetManager, world);
}

5. Game Loop (Render Cycle)

Setiap frame (60 FPS = 60x per detik), method berikut dipanggil:

// GameScreen.java
public void render(float delta) {
    // 1. Clear screen (hapus gambar sebelumnya)
    Gdx.gl.glClearColor(0, 0, 0, 1);
    Gdx.gl.glClear(GL20.GL_COLOR_BUFFER_BIT);
    
    // 2. Update game logic
    update(delta);
    
    // 3. Render world
    batch.begin();
    worldRenderer.render(batch, camera);
    batch.end();
    
    // 4. Render UI
    uiStage.draw();
}

private void update(float delta) {
    // Update player movement
    playerController.update(delta);
    
    // Check wild Pokemon encounter
    if (playerJustFinishedStep()) {
        Pokemon wild = encounterManager.checkEncounter(world, x, y);
        if (wild != null) {
            startBattle(wild);
        }
    }
    
    // Update camera dan world
    camera.update(player.getWorldX(), player.getWorldY());
    world.update(delta);
}

7. Sistem Save/Load

SaveManager.java

public class SaveManager {
    // Simpan ke LibGDX Preferences (cross-platform)
    private static final String PREFERENCES_NAME = "pokemon_save";
    private static final int MAX_SAVE_SLOTS = 3;
    
    // Data yang disimpan (GameSaveData)
    - worldName          // Nama world saat save
    - playerX, playerY   // Posisi player
    - playerFacing       // Arah hadap
    - team (List<Pokemon>) // Tim Pokemon
    - playtimeSeconds    // Waktu bermain
    
    // Method
    - saveGame(slot, data)    // Simpan ke slot
    - loadGame(slot)          // Load dari slot
    - quickSave(data)         // Save cepat (F5)
    - quickLoad()             // Load cepat (F8)
}
